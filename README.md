# 건축기사 CBT

> 건축기사 필기 기출문제를 회차별·유형별로 풀고 오답을 관리하는 웹 기반 모의고사

건축기사 필기시험(건축계획·건축시공·건축구조·건축설비·건축법규)을 대비하는 CBT(Computer-Based Test) 형식의 학습 도구다. 2016년부터 2024년까지의 기출문제 2,400문항을 담고 있으며, 회차별 모의고사와 과목별 랜덤 출제, 빈출·중요 문항 위주 출제를 지원한다. 로그인 없이 브라우저 로컬 저장소에 학습 이력과 오답을 기록하므로 접속하는 즉시 문제 풀이를 시작할 수 있다.

## 기능

- 회차별 모의고사: 연도와 회차를 골라 100문항(5과목 × 20문항)을 실제 시험 구성 그대로 응시
- 랜덤 출제: 과목과 문항 수를 선택해 원하는 분량만큼 무작위로 출제
- 빈출·중요 모드: 과목별 빈출 키워드와 최신 회차 가중치를 반영해 중요 문항 위주로 출제
- 학습 모드와 시험 모드: 선택 즉시 정답과 해설을 확인하거나, 타이머를 두고 종료 후 일괄 채점
- 오답 노트: 틀린 문항을 자동으로 모아 다시 풀고, 이해한 문항은 목록에서 제거
- 성적 분석: 응시 후 총점과 과목별 정답률, 합격 기준 통과 여부를 확인
- 다크 모드: 시스템 설정을 따르며 수동 전환도 가능

## 시작하기

요구 사항은 Node.js 20 이상이다. 의존성을 설치한 뒤 개발 서버를 실행하고 http://localhost:3000 에 접속한다.

```bash
npm install
npm run dev
```

그 밖의 명령은 다음과 같다.

```bash
npm run build      # 프로덕션 빌드
npm run start      # 빌드 결과 실행
npm run lint       # 정적 분석
npm run typecheck  # 타입 검사
npm test           # 단위 테스트
```
